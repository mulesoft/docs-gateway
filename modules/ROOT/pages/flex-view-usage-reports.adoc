== Summary

Usage reports provide insights into your monthly API calls. You can view monthly API calls by business group, environment, and registration. Access your reports via the username icon on the top right navigation bar in Anypoint Platform.

Reports are updated on the 5th day of each month.

[NOTE]
====
Usage reports are only available at the root organization level.
====

The following image illustrates an example usage report:

image:gateway-usage-report.png["Example usage report",width=95%]

== Permissions

To access usage reports, an admin must grant you the *Usage Viewer* permission in *Access Management*.
