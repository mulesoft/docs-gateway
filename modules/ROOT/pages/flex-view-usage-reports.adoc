= Viewing Flex Gateway Usage Reports
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]
:imagesdir: ../assets/images

Usage reports provide insights into your monthly API calls. You can view monthly API calls by business group, environment, and registration. Access your reports via the organization icon on the top right navigation bar in Anypoint Platform.

[NOTE]
====
Usage reports are only available at the root organization level.
====
