To view basic information about APIs associated with Flex Gateway instances:

. From Anypoint Platform, select *Runtime Manager*.
. Select *Flex Gateway* in the navigation menu.
. Click the name of the gateway whose APIs you want to check.
. Click *APIs*.
+
For each API, Runtime Manager shows the name, version, instance ID, label, and date on which it was added.

To see more detailed information and perform actions on your APIs, click *View details* to be redirected to API Manager.
