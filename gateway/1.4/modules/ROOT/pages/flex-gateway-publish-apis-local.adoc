For Flex Gateway running in Local Mode, you publish APIs using xref:flex-local-configuration-reference-guide.adoc[local declarative configuration files]. You specify the location of these configuration files when you first run Flex Gateway.

See xref:flex-local-reg-run.adoc[] for more information about how to run the gateway.
