Setting up Anypoint Flex Gateway consists of installation and registration:

* xref:flex-install.adoc[]
** Install in Linux. Download for Docker or Kubernetes.
* xref:flex-conn-reg-run.adoc[]
** Register in Connected Mode using an Anypoint username and password, a connected application, or a token.
* xref:flex-local-reg-run.adoc[]
** Register in Local Mode using an Anypoint username and password, a connected application, or a token.
