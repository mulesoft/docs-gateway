Flex Gateway enables configuration via Anypoint Platform (Connected Mode) or declarative configuration files (Local Mode). Configurations include:

* Replicas
* TLS contexts
* Shared storage
* Forward proxies
* Automated Jenkins pipelines
* Node selection for Kubernetes deployments

For Connected Mode, see xref:flex-conn-configure.adoc[].

For Local Mode, see xref:flex-local-configure.adoc[].
